# Diagrama dinámico de turnos

Pequeña demo en HTML/JS que muestra una cuadrícula tipo "contribuciones" y lista quién está trabajando hoy.

Archivos:
- [index.html](index.html)
- [style.css](style.css)
- [script.js](script.js)

Uso:

1. Abrir `index.html` en el navegador (doble clic) o servir la carpeta con un servidor local.

Para servir con Python (recomendado para evitar restricciones de CORS/paths):

```bash
python -m http.server 8000
# luego abrir http://localhost:8000
```

Personalizar la lista de personas y sus `offDays` en `script.js`.
